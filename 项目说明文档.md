宠物领养系统项目文档​
​
1. 项目概述​
这是一个基于 Java Web 的宠物领养系统，旨在为爱宠人士提供一个在线领养宠物的平台。系统分为管理员和普通用户两个角色，提供宠物信息管理、领养申请处理、用户管理等功能，搭建起宠物救助机构与领养人之间的便捷桥梁。​
2. 功能模块​
2.1 用户模块​
用户注册与登录​
查看可领养宠物列表（按状态筛选）​
查看宠物详细信息（含照片、品种、年龄等）​
提交领养申请（填写联系人信息及领养动机）​
查看个人领养记录（待审核 / 已通过 / 已拒绝）​
2.2 管理员模块​
宠物信息管理（添加、编辑、删除宠物信息，更新领养状态）​
领养申请审核（批准 / 拒绝申请，同步更新宠物状态）​
用户管理（查看注册用户列表，禁用 / 启用用户账号）​
数据统计与分析（查看领养数据、宠物分布等）​
3. 技术栈​
后端技术​
核心框架：Java Servlet（MVC 架构模式）​
数据库：MySQL（关系型数据库）​
数据访问：JDBC + DAO 模式（数据访问对象设计模式）​
Web 容器：Tomcat 9.0+​
构建工具：Maven​
前端技术​
视图层：JSP + JSTL + EL 表达式​
样式：CSS（页面美化与布局）​
客户端脚本：JavaScript（表单验证、交互效果）​

8. 部署说明​
8.1 环境准备​
JDK 8+（推荐 JDK 8，与 Tomcat 9 兼容性最佳）​
Tomcat 9.0+​
MySQL 5.7+​
Maven 3.6+（用于构建项目）​
8.2 部署步骤​
创建数据库：​
登录 MySQL，执行 SQL 语句创建数据库：create database chongwu;​
执行项目提供的完整 SQL 脚本（创建表结构并导入测试数据）​
配置数据库连接：​
找到项目中数据库连接配置文件（如 util/JDBCUtil.java 或 resources/jdbc.properties）​
修改数据库连接参数（URL、用户名、密码），确保与本地 MySQL 配置一致​
构建项目：​
进入项目根目录，执行 Maven 命令：mvn clean package​
构建成功后，在 target 目录下生成 WAR 包（pet-adoption.war）​
部署到 Tomcat：​
将 WAR 包复制到 Tomcat 的 webapps 目录下​
启动 Tomcat 服务器（执行 bin/catalina.bat run 或通过 IDE 启动）​
访问系统：​
用户端访问：`http://localhost:8080/login.jsp​
管理端访问：http://localhost:8080/adminLogin.jsp（管理员账号：admin，密码：123456）​
